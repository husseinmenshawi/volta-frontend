# project setup

1. Clone the project to your local machine using the following command:

```
git clone https://github.com/husseinmenshawi/volta-frontend.git
```

2. Install the required dependencies:

```
npm install
```

3. Run the project:

```
npm run start
```

4. Rock & roll
